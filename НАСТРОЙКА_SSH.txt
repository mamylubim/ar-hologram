Я создал скрипт для автоматической настройки!

═══════════════════════════════════════════════════════════════

🚀 БЫСТРАЯ НАСТРОЙКА:
───────────────────────────────────────────────────────────────

Просто запустите:
    ./настроить_ssh.sh

Скрипт автоматически:
1. Создаст SSH ключ (если его нет)
2. Покажет ключ для копирования
3. Подождет пока вы добавите его на GitHub
4. Настроит автоматическую загрузку

═══════════════════════════════════════════════════════════════

📋 ЧТО НУЖНО СДЕЛАТЬ:
───────────────────────────────────────────────────────────────

1. Запустите скрипт:
   ./настроить_ssh.sh

2. Скопируйте SSH ключ который покажет скрипт

3. Добавьте ключ на GitHub:
   - Зайдите на: https://github.com/settings/keys
   - Нажмите "New SSH key"
   - Вставьте скопированный ключ
   - Нажмите "Add SSH key"

4. Вернитесь в терминал и нажмите Enter

5. Готово! Теперь можно использовать:
   ./быстрая_загрузка.sh

═══════════════════════════════════════════════════════════════

💡 ПОСЛЕ НАСТРОЙКИ:
───────────────────────────────────────────────────────────────

Теперь каждый раз когда вы хотите загрузить изменения:

1. Редактируете файлы
2. Сохраняете
3. Запускаете: ./быстрая_загрузка.sh
4. Готово! Изменения на GitHub!

═══════════════════════════════════════════════════════════════

⚠️  ВАЖНО:
───────────────────────────────────────────────────────────────

- SSH ключ нужен только один раз
- После настройки все будет работать автоматически
- Не нужно вводить пароль каждый раз
