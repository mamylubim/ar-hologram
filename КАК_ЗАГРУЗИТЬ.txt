НА СТРАНИЦЕ GITHUB ЕСТЬ ИНСТРУКЦИИ!

Посмотрите на страницу - там есть раздел:
"...or push an existing repository from the command line"

Используйте эти команды в Терминале:

═══════════════════════════════════════════════════════════════

ВАРИАНТ 1: ЧЕРЕЗ ТЕРМИНАЛ (если настроена аутентификация)
───────────────────────────────────────────────────────────────

Откройте Терминал и выполните:

cd "/Users/admin/Desktop/Новая папка"
git remote add origin https://github.com/mamylubim/ar-hologram.git
git branch -M main
git push -u origin main

Если попросит логин/пароль - введите ваши данные GitHub


ВАРИАНТ 2: ЧЕРЕЗ БРАУЗЕР (проще!)
───────────────────────────────────────────────────────────────

1. На странице GitHub найдите текст:
   "Get started by creating a new file or uploading an existing file"
   
2. Нажмите на ссылку "uploading an existing file"

3. Перетащите ВСЕ файлы из папки "Новая папка"

4. Нажмите "Commit changes"


ВАРИАНТ 3: СОЗДАТЬ ФАЙЛ ВРУЧНУЮ
───────────────────────────────────────────────────────────────

1. На странице GitHub нажмите на ссылку "creating a new file"
2. Создайте файл README.md (любой текст)
3. Нажмите "Commit new file"
4. После этого появится кнопка "Add file" → "Upload files"

═══════════════════════════════════════════════════════════════

💡 САМЫЙ ПРОСТОЙ СПОСОБ:

Найдите на странице ссылку "uploading an existing file"
(она в тексте инструкций) и нажмите на неё!
