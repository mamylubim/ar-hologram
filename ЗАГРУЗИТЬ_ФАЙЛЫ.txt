ШАГ 1: ОТКРОЙТЕ РЕПОЗИТОРИЙ
───────────────────────────────────────────────────────────────

Зайдите на: https://github.com/mamylubim/ar-hologram


ШАГ 2: ЗАГРУЗИТЕ ФАЙЛЫ
───────────────────────────────────────────────────────────────

1. На странице репозитория нажмите кнопку:
   "uploading an existing file"
   (или "Add file" → "Upload files")

2. Перетащите ВСЕ файлы из папки "Новая папка":
   ✅ index-arjs.html
   ✅ video.mp4
   ✅ qr.html
   ✅ marker.html
   ✅ СОЗДАТЬ_QR.html
   ✅ README.md
   ✅ Все остальные файлы

3. Внизу страницы нажмите:
   "Commit changes" (зеленая кнопка)


ШАГ 3: ВКЛЮЧИТЕ GITHUB PAGES
───────────────────────────────────────────────────────────────

1. В репозитории перейдите: Settings → Pages

2. В разделе "Source":
   - Branch: выберите "main"
   - Folder: выберите "/ (root)"
   - Нажмите "Save"

3. Подождите 1-2 минуты


ШАГ 4: ГОТОВО!
───────────────────────────────────────────────────────────────

Ваш сайт будет доступен по адресу:
https://mamylubim.github.io/ar-hologram/index-arjs.html

Откройте СОЗДАТЬ_QR.html и создайте QR код с этой ссылкой!
