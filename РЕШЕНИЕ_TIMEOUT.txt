РЕШЕНИЕ 1: ДОБАВИТЬ ФАЙЛ .nojekyll (УЖЕ СДЕЛАНО!)
───────────────────────────────────────────────────────────────

Я создал файл .nojekyll - он отключает Jekyll и ускоряет деплой.

Теперь нужно загрузить этот файл на GitHub:

1. Зайдите на: https://github.com/mamylubim/ar-hologram
2. Нажмите "Add file" → "Create new file"
3. Название файла: .nojekyll (с точкой в начале!)
4. Оставьте содержимое ПУСТЫМ
5. Нажмите "Commit new file"

Или загрузите файл .nojekyll который я создал в папке.

РЕШЕНИЕ 2: ЕСЛИ ВСЕ РАВНО НЕ РАБОТАЕТ
───────────────────────────────────────────────────────────────

Если timeout все равно происходит, попробуйте:

1. Сжать видео (сделать меньше размером)
2. Или использовать другой хостинг для видео
3. Или загрузить видео отдельно на другой сервис

РЕШЕНИЕ 3: ПЕРЕЗАПУСТИТЬ ДЕПЛОЙ
───────────────────────────────────────────────────────────────

После добавления .nojekyll:

1. Settings → Pages
2. Измените Branch на другой
3. Save
4. Верните на "main"
5. Save
6. Подождите 5 минут
